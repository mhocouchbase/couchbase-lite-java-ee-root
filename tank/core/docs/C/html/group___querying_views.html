§ c4view_fullTextQuery()

C4QueryEnumerator* c4view_fullTextQuery ( C4View *  view,
C4Slice  queryString,
C4Slice  queryStringLanguage,
const C4QueryOptions *  c4options,
C4Error *  outError 
)

Runs a full-text query and returns an enumerator for the results.

The enumerator's fields are not valid until you call c4queryenum_next(), though.

Parameters
viewThe view to query.
queryStringA string containing the words to search for, separated by whitespace.
queryStringLanguageThe human language of the query string as an ISO-639 code like "en"; or kC4LanguageNone to disable language-specific transformations like stemming; or kC4LanguageDefault to fall back to the default language (as set by c4key_setDefaultFullTextLanguage.)
c4optionsQuery options. Only skip, limit, descending, rankFullText are used.
outErrorOn failure, error info will be stored here.
Returns
A new query enumerator. Fields are invalid until c4queryenum_next is called.
